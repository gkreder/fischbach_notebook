HSV 06-1-17

I've now written a function in analyze_groups.py that will find common clusters across all ranges of I values. I've written up a plotting function in plot_groups.py that takes in a list of clusters and bokeh plots it. I'm hoping that I can use this for a couple of different results here.

Common HHV6 genes

I've found a list of genes (6 clusters with one gene each per HHV6 species) of the clusters that are unique to HHV6. I'm spitting the output data into the folder cluster_data that's contained in the orthomcl_files folder.

I've written a function called

get_common_clusters(target_species)

That takes a list of species names and finds the clusters that only contain genes in the target_species list. It returns a list of clusters. I've also written up the plotting function above called

plot_clusters(clusters_list, output_fname = 'output.html', title = 'Title')

which is contained in the file plot_groups.py.

Producing more inflation values and plotting them

When I tried to do the same for HHV common genes (across all HHV's), I ran into the issue that there were no clusters that met the requirement that they be present in half of all Inflation value data sets. I wanted to change the value but didn't really have a good way of going about it so I produced a bunch more inflation value data sets (1-100 now). I also plotted the inflation value vs the number of clusters. Contained in file ival_line.html
